# client-project

Express 练习项目的前端部分代码。

## 1. 项目依赖

- Vue 3
- vue-router
- vuex
- element-plus@2.2.19
- axios@1.1.3
- crypto-js@4.1.1



## 2. 目录说明

```
├─public	静态资源 
└─src
│   ├─api  接口
│   ├─assets  资源
│   ├─components	组件
│   ├─router	路由
│   ├─store		vuex缓存
│   └─views		页面
│  .browserslistrc
│  .eslintrc.js
│  .gitignore
│  babel.config.js
│  package-lock.json
│  package.json
│  README.md
│  tsconfig.json
│  vue.config.js
```
